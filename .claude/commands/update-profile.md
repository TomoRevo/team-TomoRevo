# プロファイル更新コマンド

ユウジさんとの対話から得た新しい情報をもとに、オーナープロファイルとCLAUDE.mdを更新する。

## 手順

1. **現在のプロファイル確認**
   - `CLAUDE.md` を読み込み、現在のプロファイル内容を把握
   - `memory/user_yuji_profile.md` の内容も確認

2. **ユーザーへのヒアリング**
   以下を確認する（すでに会話の中で情報があればスキップ）：
   - 何を更新したいか（理念・ビジョン・目標・思考の癖・強み・事業方向性）
   - 具体的な変更内容

3. **更新の実行**
   - `CLAUDE.md` の該当セクションを編集
   - `memory/user_yuji_profile.md` を同期更新
   - 事業関連の変更は `memory/project_tomorevo_current.md` も更新

4. **変更内容の確認**
   更新した内容をユーザーに提示して確認を取る。
   ユウジさんの言葉をなるべくそのまま活かし、機械的な言い換えをしすぎない。
