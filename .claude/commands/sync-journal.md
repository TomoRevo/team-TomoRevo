# ジャーナル同期コマンド

Notionジャーナルデータベースから最新エントリを取得し、CLAUDE.mdとメモリを更新する。

## 手順

1. **最新ジャーナルの取得**
   - Notion MCP `notion-search` でジャーナルDB（`collection://31f48561-540f-80db-ac7c-000bdfc3b66e`）を検索
   - 直近1週間分のエントリを取得
   - `notion-fetch` で各ページの全文を取得

2. **分析・抽出**
   以下の観点でジャーナル内容を分析する：
   - **新たな気づき・価値観の変化**: 人生理念や信念に関する新たな言語化があれば抽出
   - **思考の癖の出現パターン**: 後回し癖・完璧主義・疑心暗鬼が出ていたか、克服できたか
   - **グッドポイント**: できたこと・前進したことを一覧化
   - **現在のフォーカスの変化**: 目標や優先事項に変更があれば検出
   - **新しい強み・スキル**: 自覚した新たな強みがあれば記録

3. **CLAUDE.mdの更新**
   - `CLAUDE.md` の「現在のフォーカス」セクションを最新の状態に更新
   - 思考の癖や価値観に重要な変化があれば該当セクションも更新
   - 変更点をユーザーに報告

4. **メモリの更新**
   - `memory/user_yuji_profile.md` に新たな気づきがあれば追記
   - `memory/project_tomorevo_current.md` の事業フォーカスに変化があれば更新

5. **レポート出力**
   以下のフォーマットで同期結果を報告する：

   ```
   ## ジャーナル同期レポート（YYYY/MM/DD）

   ### 取得したエントリ
   - YYYY/MM/DD: （タイトルまたは概要）

   ### 検出した変化
   - （価値観・目標・思考パターンの変化点）

   ### 更新した箇所
   - （CLAUDE.mdやメモリの変更点）

   ### グッドポイント
   - （ジャーナルから読み取れた前進・成長のポイント）
   ```
