# 週次レビューコマンド

1週間分のジャーナルとタスク実績を総合的にレビューし、成長と課題を可視化する。

## 手順

1. **データ収集**
   - Notionジャーナルから直近7日分のエントリを取得（`collection://31f48561-540f-80db-ac7c-000bdfc3b66e`）
   - Notionタスク管理から完了タスクを取得（`3fb08cb1-64e8-42bc-96fe-1dacf983f399`）
   - Google Sheetsからタスク実績を取得（`1faOIQzkisQzlgBl_VhxB5iO-kfkTb_Z9_x4G4n8VXS8`）

2. **分析**
   以下の軸で1週間を分析する：

   **a) 約束の履行度**
   - 人との約束：守れた / 守れなかった
   - 自分との約束：日課チェックリスト、振り返り、ジャーナルの継続状況

   **b) 思考の癖トラッキング**
   - 後回し癖が出た場面 / 克服できた場面
   - 完璧主義が出た場面 / うまく手放せた場面
   - 疑心暗鬼が出た場面 / 素直に受け取れた場面

   **c) 理念との整合性**
   - 「愛・感謝・ワクワク」に沿った行動ができたか
   - ビジョン「子供が大人に憧れる社会を作る」に近づく行動はあったか

   **d) 事業の進捗**
   - トモレボ / AIコンサル / SNS発信の進捗

3. **レポート出力**
   以下のフォーマットで報告：

   ```
   ## 週次レビュー（MM/DD 〜 MM/DD）

   ### 今週のグッドポイント
   1.
   2.
   3.

   ### 約束の履行スコア
   - 人との約束: ○/△/× （具体例）
   - 自分との約束: ○/△/× （具体例）
   - ジャーナル継続: ○日/7日

   ### 思考の癖レポート
   | 癖 | 出た場面 | 克服できた場面 |
   |----|---------|--------------|
   | 後回し | ... | ... |
   | 完璧主義 | ... | ... |

   ### 理念チェック
   - 愛: （具体的なエピソード）
   - 感謝: （具体的なエピソード）
   - ワクワク: （具体的なエピソード）

   ### 来週に向けて
   - （1つだけ、最も効果的な改善アクションを提案）
   ```

4. **CLAUDE.md更新**
   - 「現在のフォーカス」に変化があれば更新
   - 新しい強みや気づきがあればプロファイルに追記
