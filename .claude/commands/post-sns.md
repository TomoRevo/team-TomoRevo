# SNS毎日投稿コマンド

STAY GOLD天満店のSNS毎日投稿（X / Threads）を実行する。

## 前提情報
- 投稿カレンダー: 2026年3月13日〜4月11日（30日分）
- Day 1 = 3/13, Day 2 = 3/14, ..., Day 30 = 4/11
- Notionソース:
  - X投稿: `https://www.notion.so/32248561540f819e810cd0e32f979b42`
  - Threads投稿: `https://www.notion.so/32248561540f81f6ac52e8ab2836082c`
  - MEO投稿: `https://www.notion.so/32248561540f81bbb302f7f8a6dd3bd5`
- 投稿管理DB: `collection://d94779d8-f4bc-4887-9610-42c43853de0d`
  - DB URL: `https://www.notion.so/4e6676ee0514422e9994621768253ad6`

## 手順

1. **今日のDay番号を特定**
   - 今日の日付から Day番号を計算（3/13起算）
   - 投稿管理DBで該当日の投稿ステータスを確認

2. **投稿コンテンツを取得**
   - Notionから該当Dayの X / Threads の投稿本文を取得
   - ツリー形式（複数投稿）の場合はすべて取得

3. **投稿内容をユーザーに確認**
   - X投稿とThreads投稿の内容を表示
   - ユーザーが「OK」と言ったら投稿に進む
   - 修正が必要な場合は修正して再確認

4. **X（旧Twitter）に投稿**
   - Playwrightで https://x.com にアクセス
   - ログイン状態を確認（未ログインならログインを依頼）
   - 投稿画面を開いて本文を入力・投稿
   - ツリー形式の場合は1投稿目→返信として2投稿目を投稿
   - 投稿完了を確認

5. **Threadsに投稿**
   - Playwrightで https://www.threads.net にアクセス
   - ログイン状態を確認（未ログインならログインを依頼）
   - 「作成」ボタンから投稿画面を開いて本文を入力・投稿
   - ツリー形式の場合は返信で追加投稿
   - トピックタグ（ハッシュタグ）を設定
   - 投稿完了を確認

6. **投稿管理DBを更新**
   - 該当日のX投稿済み・Threads投稿済みのチェックを入れる
   - 投稿日時を記録

7. **完了報告**
   - 投稿した内容のサマリーと投稿URLを報告
