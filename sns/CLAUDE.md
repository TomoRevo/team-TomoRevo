# STAY GOLD SNSマーケティング

## 概要
パーソナルジム「STAY GOLD」のSNSマーケティング戦略・コンテンツ制作を管理するフォルダ。
Instagram → LINE → 個別相談/ジム体験へのファネルを構築中。

## ファネル全体像
```
Instagram Reels（認知・集客）
↓
LINE友だち追加
↓
ダイエット診断（5問）
↓
タイプ別結果 + 7日間チャレンジ開始（DAY1〜7 動画+テキスト配信）
↓
無料オンライン個別相談 or ジム体験予約
```

## フォルダ構成
```
sns/
├── CLAUDE.md                          ← このファイル
├── README.md                          ← 進捗管理
├── LINE戦略/
│   ├── 7日間チャレンジ_メッセージ設計.md  ← 全メッセージ文面・配信設計
│   └── 実装計画_システム設計.md           ← 技術構成・作業工程
├── コンビニ食リサーチレポート.md           ← 高たんぱく低脂質商品の調査結果
└── おすすめコンビニ食/                    ← LINE特典PDF制作ツール
    ├── create_horizontal_pdf.py         ← PDF生成スクリプト
    ├── product_images/                  ← 商品画像
    └── output/                          ← 生成PDF
```
