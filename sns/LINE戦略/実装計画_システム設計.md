# LINE ファネル実装計画書

最終更新：2026-03-10

---

## プロジェクト概要

**ゴール**
- 最重要：オンライン無料個別相談の予約
- 次点：大阪パーソナルジム体験予約

**ファネル**
Instagram → LINE登録 → 診断 → 7日チャレンジ → 相談予約

---

## システム構成

```
Instagram Reels
↓ プロフィールリンク
友だち追加LP（Web）
↓ LINE友だち追加
LINE公式アカウント（Lステップ連携）
↓
診断ツール（LIFF or Web）
↓
Lステップ ステップ配信（DAY1〜7）
↓
相談フォーム（Lステップ）
↓
管理者通知
```

---

## 技術選定

| カテゴリ | 技術 | 備考 |
|---|---|---|
| LINE配信 | Lステップ（Lメッセージ） | ステップ配信・タグ・フォーム |
| 診断ツール | Next.js + LIFF | LINE内で動作 |
| フロントエンド | Next.js / Tailwind CSS | - |
| ホスティング | Vercel | 無料枠で対応可 |
| データ管理 | Google Sheets | 管理画面不要・エクスポート容易 |

---

## Lステップ設定内容

### シナリオ設計
1. 友だち追加時シナリオ → ウェルカム + 診断誘導
2. 診断完了タグ付与 → タイプ別ステップ配信開始
3. DAY1〜7 ステップ配信
4. DAY7完走タグ付与 → オファー送信

### タグ設計
| タグ名 | 付与タイミング |
|---|---|
| 診断済み | 診断ツール完了時 |
| タイプ①〜④ | 診断結果確定時 |
| DAY完走 | DAY7終了時 |
| 相談申込済み | フォーム送信時 |
