最終更新：2026-03-16

## この動線のゴール

**体験セッション（ジム or オンライン）の予約を取ること。**

ただし大前提として：
> この特典だけでも、受け取った人の人生が前進するきっかけになること。
> 「売るための撒き餌」ではなく、「本気で届けたいもの」であること。

これはユウジさんの信念であると同時に、最も合理的な戦略でもある。
本気で届けたものだけが、本気の信頼を生む。

## ユーザーの感情ジャーニー

特典動線の全体を通して、この感情の流れを設計する：

```
① 好奇心 →「私って何タイプなんやろ？」
     ↓
② 自己発見 →「あ、だから今まで上手くいかなかったんや」
     ↓
③ 希望 →「自分に合うやり方があるんや」
     ↓
④ 行動 →「これならできそう、やってみよう」
     ↓
⑤ 実感 →「体が変わり始めてる…！」
     ↓
⑥ 期待 →「もっと変われるかも。この人と一緒にやったらどうなるんやろ」
```

**この⑥の感情が自然に湧いた時に、体験セッションのCTAが目の前にある。**
これが理想の動線。

### 2. 診断ページ（LINE登録不要）

**目的：好奇心を満たしながら、心理的投資を作る**

現状の5問はタイプ分類としては機能しているが、
ユーザーの感情ジャーニー①②（好奇心 → 自己発見）を強化するために、
**質問の切り口を「自分を知る体験」に寄せる。**

#### 診断5問（改善案）

**Q1. ダイエットで一番つらいのは？**
- A. そもそも何から始めればいいかわからない
- B. 食事を我慢すること
- C. 運動する時間を確保すること
- D. 頑張っているのに結果が出ないこと

→ 「つらいこと」から入ると「わかってくれてる」感が出る

**Q2. 過去のダイエット、どうだった？**
- A. ほぼやったことがない
- B. 食事制限で一時的に痩せたがリバウンドした
- C. 何度か挑戦したが途中で忙しくなって中断
- D. いろいろ試したが、どれも続かなかった

→ 失敗体験に共感 → 「だからこそ、あなたに合う方法がある」の伏線

**Q3. 理想の体型になれたら、一番やりたいことは？**
- A. 好きな服を自信を持って着たい
- B. 食事を気にせず楽しみたい
- C. 体が軽くなって毎日アクティブに過ごしたい
- D. 自分に自信を持って、人前で堂々としていたい

→ ここで「未来のワクワク」を自分で言語化させる（重要）

**Q4. 1日の中で自分のために使える時間は？**
- A. 正直、ほとんどない（5分くらい）
- B. 10〜20分ならなんとか
- C. 30分〜1時間はある
- D. 1時間以上確保できる

→ 現実的な制約を把握

**Q5. どんなサポートがあったら続けられそう？**
- A. 何も考えなくていい、言われた通りにやるだけのプログラム
- B. 食事の「これだけやればOK」リスト
- C. 短時間でできる、スキマ時間プログラム
- D. 自分の状態を見てくれる、マンツーマンのサポート

→ 最後の質問で「体験セッション」への伏線を張る（特にD）

#### タイプ判定ロジック

| 回答傾向 | タイプ |
|----------|--------|
| A多め | はじめの一歩タイプ |
| B多め | 食事リセットタイプ |
| C多め | スキマ時間活用タイプ |
| D多め | 本気スイッチタイプ |

**タイプ名の変更理由：**
- 「超初心者」→「はじめの一歩」：ネガティブラベルを避ける。前向きな響き
- 「食事改善」→「食事リセット」：「改善」は上から目線。「リセット」はフレッシュスタート感
- 「忙しい」→「スキマ時間活用」：問題ではなく解決策をタイプ名にする
- 「本気」→「本気スイッチ」：やる気がある前提で、あとは「正しい方向」だけという示唆

### 4. タイプ別PDFガイド（LINE登録直後に送付）

**目的：「この特典だけでも人生が前進する」を実現する**

受け取った瞬間に「うわ、これ無料でいいの？」と思わせる。
同時に「この人にもっと教えてもらいたい」という感情を醸成する。

#### PDF構成（各タイプ共通フォーマット / 8〜10ページ）

| ページ | 内容 | 感情設計 |
|--------|------|----------|
| 1 | 表紙「あなたは○○タイプ」 | 特別感 |
| 2 | このタイプの特徴 | 「わかってくれてる」 |
| 3 | 今まで上手くいかなかった理由 | 「だからか！」という気づき |
| 4 | あなたに合うダイエットの3つの鍵 | 希望（合う方法がある） |
| 5-6 | 今日からできるアクション（具体的・実践的） | 「これならできる」 |
| 7 | あなたの3ヶ月後のイメージ | ワクワクする未来の提示 |
| 8 | ユウジからのメッセージ | 人柄・信頼 |
| 9 | 「もっと本気で変わりたくなったら」 | 体験セッションへの案内 |

#### 各タイプのPDF内容方針

**はじめの一歩タイプ**
- 3つの鍵：①完璧を捨てる ②「3分だけ」を毎日やる ③体重計より体感で測る
- アクション：朝起きたら1曲踊る / 水を1杯多く飲む / 寝る前に今日の自分を褒める
- 3ヶ月後：「体が軽い」が当たり前になっている。階段を避けなくなる。好きな服に手が伸びる

**食事リセットタイプ**
- 3つの鍵：①「食べない」ではなく「選ぶ」 ②コンビニは味方 ③食後10分の動きが全てを変える
- アクション：昼食を1品だけ変える / 間食をギリシャヨーグルトに / 食後に1曲踊る
- 3ヶ月後：食べることへの罪悪感がなくなっている。「食べて痩せる」が自分の中で当たり前になる

**スキマ時間活用タイプ**
- 3つの鍵：①「時間がない」は思い込み ②3分 × 3回 = 9分で十分 ③「ながら運動」を味方にする
- アクション：通勤中に1駅歩く / 歯磨き中にスクワット / 寝る前に1曲踊る
- 3ヶ月後：「忙しいから無理」と言わなくなっている。体を動かすことが生活に溶け込んでいる

**本気スイッチタイプ**
- 3つの鍵：①方向性の修正が最優先 ②「頑張る量」ではなく「正しい順番」 ③メンタルと体はセット
- アクション：今やっていることを1つ「やめる」/ 毎朝体の状態を10秒観察する / 集中して取り組む
- 3ヶ月後：「頑張っているのに変わらない」から卒業している。努力が結果に直結する感覚を取り戻している

#### p.7「3ヶ月後のイメージ」の書き方ガイドライン

ここが最も重要なページ。「うわ、こんな未来が待ってるんか」を感じさせる。

- 抽象的な理想ではなく、**日常の具体的なシーン**で描く
- 「痩せた自分」ではなく「痩せた結果、何ができるようになっているか」
- 例：
  - × 「理想の体型を手に入れている」
  - ○ 「鏡の前で、去年は入らなかったスカートがすっと入る」
  - ○ 「友達と温泉に行くのが怖くなくなっている」
  - ○ 「子供と全力で走れる自分がいる」

### 5. ステップ配信

#### Version A：動画なし（一括配布 + 限定オファー）

温度が一番高いのはLINE登録直後。分割配信で冷ます理由がない。
全部まとめて渡し、熱いうちにオファーする。

| タイミング | 内容 | 感情設計 |
|-----------|------|----------|
| 登録直後 | ウェルカム + 全特典まとめて送付（タイプ別PDF / コンビニ食材ガイド / ダイエット教科書） | 「こんなにもらえるの？」約束の即履行 |
| 3〜4時間後 | お客様の声・変化の実例 + 体験セッションオファー（3日間限定） | 「この人のサポートで変われるんや」 |
| 翌日（24h後） | リマインド①「特典、見てもらえましたか？」+ 体験の空き状況 | 温度の再加熱 |
| 3日目（48h後） | リマインド②「明日で無料体験の受付を締め切ります」 | 限定性 |
| 4日目（72h後） | 締切通知「本日で締め切りました。また機会があれば案内します😊」 | 関係は切らない |

**合計：5配信**

**Version Aの狙い：**
- 登録直後の温度MAXで全特典を渡す（出し惜しみしない）
- 特典を読む時間を3〜4時間確保してからオファー
- 3日間の限定性で行動を促す（煽りではなく「枠が限られている」事実を伝える）
- 締切後も関係を切らない（将来の再オファーに繋げる）

**限定性の伝え方（愛起点と両立させる）：**
- × 「今すぐ申し込まないと損します！」（煽り）
- ○ 「体験セッションは週に○枠しか取れないので、3日間だけ受け付けています」（事実）

ユウジさんの原理原則に立ち返ると：

> **「愛 × 仕組み = お金」**

- 特典の中身（PDF、動画、メッセージ）= **愛**（本気で届けたいもの）
- ファネルの構造（診断→LINE→配信→体験）= **仕組み**
- この両輪が揃った時に、結果として体験予約（お金）が生まれる

特典を「撒き餌」にした瞬間、愛が消えて仕組みだけになる。
だからこそ「この特典だけでも人生が前進する」が最重要の設計原則。
