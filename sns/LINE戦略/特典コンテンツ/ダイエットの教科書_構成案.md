## 第1章：うまくいく人が最初に知っている3つのこと（1ページ）

**【意図】** 読者が「自分のことだ」と感じる入口を作る。よくあるつまずきポイントを3つに絞り、「やり方を変えるだけでうまくいく」と前向きな気持ちにさせることで、続きを読むモチベーションを作る。

### 構成

**リード文**
ダイエットがうまくいく人は、実はちょっとしたコツを知っているだけ。
やり方を変えるだけで、体は応えてくれる。まずは「うまくいく人が押さえている3つのポイント」を見てみてください。

**ポイント1：食べ方を変えるだけで体は変わる**
- 食事量を極端に減らすと、体は「飢餓状態」と判断して省エネモードに入る
- 何もしなくても体が消費するエネルギー（基礎代謝）が下がる
- 結果として、食べる量を戻した途端にリバウンドする
- 生活シーン例：「1日1食にしてるのに体重が落ちない…」→ これは体が省エネモードに入っているサイン。食べ方を変えるだけで、ここから変わっていける

**ポイント2：運動の「本当の役割」を知ると、効率が上がる**
- 30分のジョギングで消費するカロリーは、おにぎり約1個分（約200kcal）
- 実は、運動の本当の役割は「筋肉を守り、代謝を維持すること」
- この視点を持つだけで、運動の効果がぐっと上がる

**ポイント3：「仕組み」で続ける人がうまくいく**
- 意志力は有限。だからこそ、仕組みで続けるのがコツ
- 「続けられる仕組み」を作ることが大事
- ダイエットは短距離走ではなく、歩いて続けるもの

## 第2章：ダイエットの全体像（1ページ）

**【意図】** この教科書の全体マップを1枚で見せる。「食事・運動・習慣・メンタル」の4本柱を俯瞰することで、自分に足りないものが何かが一目でわかる。バラバラだった知識が整理される感覚を与える。

### 構成

**リード文**
ダイエットは「食事」「運動」「習慣」「メンタル」の4本柱で成り立っている。
4つをバランスよく押さえれば、無理なく体は変わる。

**4本柱の図解（ビジュアルで表現）**

```
┌──────────────────────────────────────┐
│        理想の体型・健康            │
├────────┬────────┬────────┬──────────┤
│  食事  │  運動  │  習慣  │ メンタル │
│        │        │        │          │
│ 何を   │ どう   │ どう   │ どう     │
│ 食べるか│ 動くか │ 続けるか│ 考えるか │
└────────┴────────┴────────┴──────────┘
```

**各柱の一言説明**
- **食事**: 体を作る材料。「何を食べるか」を意識するだけで変わる
- **運動**: 代謝を守り、体のラインを整える手段。短時間でOK
- **習慣**: 「仕組み」で自然に続く状態を作る
- **メンタル**: 「やりたい」から動く力を作る。自分を認めることが土台

**メッセージ**
この教科書では、4つの柱をそれぞれ解説していく。
全部を一気にやる必要はない。まずは気になる章から読んでみてください。

## 第3章：食事編 — 食べて痩せる仕組み（3ページ）

**【意図】** 食事はダイエットの最重要要素。「食べ方を変える」という発想の転換を促す。3つのテーマ（栄養バランス・血糖値・実践テクニック）に分けて、知識→具体行動までつなげる。コンビニ活用術は忙しい30〜50代女性にとって即実践できるポイント。

### 3-1. 栄養バランスの基礎（1ページ）

**リード文**
体を変えるために必要な栄養素は、大きく3つ。
タンパク質・脂質・炭水化物。この3つのバランスを意識するだけで、食事の質は大きく変わる。

**タンパク質 — 体を作る材料**
- 筋肉、肌、髪、爪の材料になる
- ダイエット中にしっかり摂ることで筋肉を守り、代謝を維持できる
- 1日の目安：体重1kgあたり約1〜1.5g（体重55kgなら55〜82g）
- イメージ：手のひら1枚分の肉・魚を毎食
- 具体例：鶏むね肉100g → タンパク質約22g / 卵1個 → 約6g

**脂質 — 必要な栄養素。量と質がポイント**
- ホルモンの材料、肌のうるおいに必要
- 少量でカロリーが高い（1gあたり9kcal。タンパク質・炭水化物は4kcal）ので、量を意識するのがコツ
- 「良い脂質」を選ぶ：魚の油（オメガ3）、オリーブオイル、ナッツ類
- 揚げ物、菓子パン、スナック菓子は控えめにすると効果的

**炭水化物 — 味方にすると心強い**
- 脳と体のエネルギー源。適度に摂ることで集中力もキープできる
- 「選ぶ」意識を持つだけで変わる：白米→玄米や雑穀米、パン→全粒粉パン
- 活動量が多い日は多めに、少ない日は控えめに調整

**バランスの目安（1食のイメージ）**
- お皿の半分 → 野菜・きのこ・海藻
- お皿の4分の1 → タンパク質（肉・魚・卵・大豆）
- お皿の4分の1 → 炭水化物（ご飯・パン・麺）
- 少量の良い脂質（オリーブオイル、ナッツなど）

### 3-2. 血糖値コントロール（1ページ）

**リード文**
「食べた後に眠くなる」「間食がやめられない」
実はこれ、意志力の話ではなく、血糖値の急上昇・急降下が原因かもしれない。
仕組みを知ると、コントロールがぐっと楽になる。

**血糖値の仕組み（シンプルに）**
- 食事をすると血糖値が上がる → 体はインスリンを出して血糖値を下げる
- 血糖値が急に上がると → インスリンが大量に出る → 血糖値が急降下 → 空腹感・眠気・イライラ
- さらに、余ったエネルギーは脂肪として蓄えられる

**血糖値が急に上がりやすい食べ物**
- 白い炭水化物（白米、食パン、うどん、パスタ）
- 甘い飲み物（ジュース、甘いカフェラテ）
- お菓子、菓子パン

**血糖値をゆるやかにする食べ方 3つのコツ**

1. **食べる順番を変える**
   - 野菜・汁物 → タンパク質 → 炭水化物の順番
   - 野菜の食物繊維が、糖の吸収をゆるやかにしてくれる
   - 生活シーン例：「ランチの定食。まずサラダと味噌汁から食べ始めるだけでOK」

2. **炭水化物を"選ぶ"**
   - 白米 → 玄米・雑穀米、食パン → 全粒粉パン、うどん → そば
   - 精製されていない穀物は食物繊維が多く、血糖値がゆるやかに上がる

3. **こまめにエネルギーを補給する**
   - 朝食を軽くでも摂ることで、昼のドカ食いを防げる
   - 生活シーン例：「朝ごはんにトーストだけだと、10時ごろにお菓子に手が伸びがち。トーストにゆで卵を1つ足すだけで、昼まで持つようになる。これだけで1日の流れが変わる」

### 3-3. 実践テクニック：コンビニ活用術（1ページ）

**リード文**
自炊が理想だけど、毎食作るのは大変。
コンビニでも「選び方」を知っていれば、ダイエット向きの食事は十分できる。

**コンビニで選びたいもの**
- サラダチキン（タンパク質が豊富、脂質が少ない）
- ゆで卵（手軽なタンパク質源）
- ギリシャヨーグルト（タンパク質が普通のヨーグルトの約2倍）
- 雑穀おにぎり（白米より食物繊維が多い）
- カット野菜・サラダ（食物繊維で血糖値をゆるやかに）
- 味噌汁・スープ（満腹感が出る。具だくさんがベスト）

**おすすめの組み合わせに変えるだけで変わる**
- 菓子パン + 甘いカフェラテ → サラダチキン + 雑穀おにぎり + サラダに変えるだけで大きく違う
- カップ麺 + おにぎり → 具だくさんスープ + ゆで卵 + おにぎりにすると、タンパク質もしっかり摂れる

**おすすめ組み合わせ例**
- 昼食：サラダチキン + 雑穀おにぎり + カット野菜サラダ（約450kcal）
- 間食：ギリシャヨーグルト + ナッツ少量（約200kcal）
- 夕食：具だくさんスープ + ゆで卵2個 + サラダ（約350kcal）

## 第4章：運動編 — 最小限で最大効果を出す方法（2ページ）

**【意図】** 「運動=つらい、時間がかかる」というイメージを変える。忙しい30〜50代女性が「これならできるかも」と思えるハードルの低さを重視。筋トレの必要性を理論で納得させてから、超短時間の実践メニューを提示する。

### 4-1. 筋トレが味方になる理由（1ページ）

**リード文**
「痩せたいからとりあえず走ろう」——実は、もっと効率の良い方法がある。
ダイエットで本当に大事なのは、筋肉を味方につけること。

**筋肉が増えると痩せやすくなる理由**
- 何もしなくても体が消費するエネルギー（基礎代謝）の約20〜30%は筋肉が使っている
- 筋肉が増える → 基礎代謝が上がる → 同じ量を食べても消費されるエネルギーが増える → 痩せやすい体になる
- 30代以降、筋肉は年に約0.5〜1%ずつ自然に減る（サルコペニアの進行）
- だからこそ、筋トレで筋肉を守ることが年々大きな差になる

**有酸素運動と筋トレの違い**

| | 有酸素運動（ウォーキング等） | 筋トレ |
|--|--|--|
| 運動中のカロリー消費 | 多い | 少なめ |
| 運動後のカロリー消費 | すぐ戻る | 数時間〜24時間続く |
| 筋肉への効果 | 維持しにくい | 維持・増加できる |
| 基礎代謝への影響 | 変化が出にくい | 上がる |

- 有酸素運動は「運動している間」カロリーを消費する
- 筋トレは「運動後も代謝が高い状態が続く」（EPOC：運動後過剰酸素消費）
- さらに筋肉量が増えれば、何もしていない時間のカロリー消費も増える

**結論**
- 有酸素運動も良い効果がある。組み合わせるとさらにベスト
- 「どちらか一つ」なら筋トレを優先する方が長期的に効果的
- 理想は「筋トレ + 日常の活動量を増やす（階段を使う、一駅歩くなど）」

### 4-2. 1日3〜5分でできること（1ページ）

**リード文**
「ジムに通う時間がなくても大丈夫」「運動が苦手でも大丈夫」。
まずは1日3〜5分でいい。大事なのは、やること自体ではなく「続くこと」。

**自宅でできる3種目（器具なし）**

**1. スクワット（下半身全体）— 体で一番大きな筋肉を使う**
- 足を肩幅に開き、椅子に座るイメージでお尻を後ろに引く
- 膝がつま先より前に出すぎないように意識する
- 10回 × 2セット（慣れたら3セット）
- ポイント：下半身の筋肉は体全体の約60〜70%を占める。ここを鍛えると効率が良い

**2. プランク（体幹）— お腹まわりを引き締める**
- うつ伏せの状態から、ひじとつま先で体を支える
- 頭からかかとまで一直線をキープ
- 20秒キープ × 2セット（慣れたら30秒）
- ポイント：呼吸を続ける。お腹に軽く力を入れる意識

**3. ヒップリフト（お尻・裏もも）— ヒップアップ効果**
- 仰向けに寝て、膝を立てる
- お尻を天井に向かって持ち上げる → 3秒キープ → ゆっくり下ろす
- 10回 × 2セット
- ポイント：お尻の筋肉を使っている感覚を意識する

**所要時間の目安**
- スクワット10回×2（約1分）+ プランク20秒×2（約1分）+ ヒップリフト10回×2（約1分）
- 合計：約3〜5分

**ストレッチの重要性**
- 運動後に30秒〜1分のストレッチを入れる
- ストレッチで筋肉をほぐすと血流が良くなり、代謝も上がりやすくなる
- 柔軟性が上がると可動域が広がり、同じ運動でも効果が高くなる
- おすすめ：前もものストレッチ、お尻のストレッチ、胸を開くストレッチ

## 第5章：習慣編 — 続けるための仕組み（2ページ）

**【意図】** 多くの人は「何をすればいいか」は知っている。問題は「続けられるかどうか」。ここでは意志力に頼らず、行動科学・習慣研究の知見を使って「仕組みで続ける」方法を伝える。80点主義は、完璧主義で動けなくなる読者への処方箋。

### 5-1. 仕組みで続く人になれる（1ページ）

**リード文**
「3日坊主なんです」「いつも続かなくて…」
実はこれ、性格の問題ではなく、仕組みの問題。仕組みを変えるだけで、自然に続く自分になれる。

**意志力は消耗する**
- 意志力は筋肉と同じで、使えば疲労する（自我消耗：心理学者ロイ・バウマイスターの研究）
- 朝は意志力が満タン → 夜になるほど減っていく
- 仕事や家事で意志力を使い果たした後に「さあ運動しよう」は最もハードルが高い
- 生活シーン例：「仕事で疲れた夜、冷蔵庫のアイスに手が伸びる。これは意志力の残量がゼロになっているだけ。仕組みを変えれば、ここから変わっていける」

**意志力に頼らず続く3つの仕組み**

**1. 行動のハードルを下げる**
- 「30分運動する」ではなく「運動ウェアに着替える」をゴールにする
- ヨガマットを出しっぱなしにする。ダンベルをテレビの横に置く
- 「やり始めたら続く」性質（作業興奮）を利用する

**2. 既存の習慣にくっつける（習慣のスタッキング）**
- 「歯磨きの後にスクワット10回」「お風呂の前にストレッチ」
- すでに毎日やっていることの前後に新しい行動を足す
- 新しい習慣を「いつやるか」を決めておくことで、判断コストがなくなる

**3. 環境を整える**
- お菓子は目につかない場所へ移す（目に入ると意志力を消耗する）
- 冷蔵庫の手前にゆで卵やカット野菜を置く
- 寝る前に運動ウェアを枕元に置いておく
- 「正解を手の届くところに置く」がルール

### 5-2. 80点主義のすすめ（1ページ）

**リード文**
100点を目指すと止まりやすい。80点で十分。
大事なのは、100点を1日やることより、70点を30日続けること。

**完璧主義を手放すと、うまくいく3つの理由**

1. **「まあいいか」が最強の味方になる**
   - 「今日はお菓子を食べちゃった → まあいいか、夕食でバランスを取ろう」
   - 1回の寄り道を、次の良い選択につなげるのがコツ
   - 現実：1日くらい食べすぎても、体脂肪はほぼ変わらない（脂肪1kgは約7,200kcal）

2. **「まず1つだけ」で動き出せる**
   - 完璧な食事プラン、完璧な運動メニューを作ろうとすると、なかなか始められない
   - 「まず1つだけやる」の方がずっと効果的

3. **自分のペースが一番の近道**
   - SNSで見る「2ヶ月で-10kg」は極端な例。健康的な減量ペースは月1〜2kg
   - 自分のペースで進んでいることを認めてあげる。それが一番の近道

**80点の具体例**
- 3食のうち2食をバランスよく食べたら80点
- 週に3日運動できたら80点
- 夜にお菓子を食べた日でも、朝と昼は意識できていたら80点
- 今日サボっても、明日また始めたら80点

**キーメッセージ**
完璧を目指さなくて大丈夫。昨日の自分より1つだけ良い選択ができたら、それで十分。

## 第6章：メンタル編 — ダイエットが続く人の考え方（2ページ）

**【意図】** 食事や運動の知識があっても、メンタルが整っていなければ続かない。ここではコーチングと選択理論心理学のエッセンスを、用語を使わずに伝える。「外からの強制ではなく、内側からの動機で人は変わる」という核心を届ける。自分を認めるサイクルを育て、「やりたい」から始める力を育てる章。

### 6-1. 自分を認めることが土台になる（1ページ）

**リード文**
ダイエットがうまくいく人には共通点がある。
それは、自分を認めることが上手なこと。「今日もがんばった」——この一言が、実は一番の味方になる。

**自分を認める → 前に進めるサイクル**
- 小さな成功を認める → 自信がつく → 次の行動が生まれる → また成功する → さらに自信がつく
- このサイクルが回り始めると、ダイエットは自然と楽しくなっていく
- 逆に、自分を責めるとストレスがたまり、ストレスホルモン（コルチゾール）が慢性的に高くなると、食欲が増し、脂肪を溜め込みやすくなる（特にお腹まわり）

**「できたこと」に目を向ける**
- 1日の終わりに「今日できた良いこと」を1つだけ書き出す
- 「3食のうち1食はバランスを意識できた」「エレベーターの代わりに階段を使った」
- 小さな成功体験の積み重ねが、自信になる
- 自信がつくと、行動が変わる。行動が変わると、結果が変わる

**体重計との付き合い方**
- 体重は1日の中で1〜2kg変動する（水分量、食事のタイミングなど）
- 毎日の数字に一喜一憂するより、1週間・2週間の平均で見るのがおすすめ
- 体重だけでなく、体のラインや服のフィット感、体調の変化にも注目する

### 6-2.「やりたい」から始める（1ページ）

**リード文**
「痩せたい」「もっと元気になりたい」——自分の内側から湧いてくる「やりたい」が、一番長く続くエネルギーになる。

**外からの力 vs 内側からの力**
- 「医者に言われたから」「周りの目が気になるから」→ 外からの力（外発的動機）
- 「自分に自信を持ちたい」「子どもと元気に遊びたい」「好きな服を着たい」→ 内側からの力（内発的動機）
- 外からの力は短期的には効くが、プレッシャーがなくなると行動も止まりやすい
- 内側からの力は、誰に言われなくても自分から動ける。だから長く続く

**あなたの「本当の理由」を見つける**
- 「なぜ痩せたいのか？」を3回掘り下げてみる

```
例：
「痩せたい」 → なぜ？
「見た目に自信を持ちたい」 → なぜ？
「自分のことを好きになりたい」 → なぜ？
「自分を好きでいられたら、毎日がもっと楽しくなるから」
```

- この一番深い理由が、あなたの「本当の動機」
- つらくなった時、この理由に立ち戻ることで、また前に進める

**「やるべき」を「やりたい」に変換してみると…**
- 「運動すると、夜ぐっすり眠れて朝が気持ちいい」
- 「タンパク質を摂ると、甘いものへの欲求が減って楽になる」
- 自分にとっての「いいこと」として捉え直すと、自然と続けたくなる
- 人は自分で選んだことには前向きに取り組める

**キーメッセージ**
ダイエットは「なりたい自分に近づくこと」。
その過程を楽しめた時、結果は自然とついてくる。

## 第7章：あなたの「次の一歩」（1ページ）

**【意図】** 読了後の行動を明確にする。タイプ別ガイドへの誘導と、体験セッションへの自然な導線を作る。売り込みではなく「もっと知りたい人はどうぞ」というスタンス。

### 構成

**リード文**
ここまで読んだあなたは、ダイエットの全体像をつかんでいる。
次は「あなたに合ったやり方」を見つける番。

**ステップ1：自分のタイプを知る**
- LINE診断であなたの「ダイエットタイプ」がわかります
- タイプに合わせた具体的なアドバイスをお届けします
- （※タイプ別ガイドへのリンク or 案内）

**ステップ2：まず1つだけ、今日やってみる**
- 今日の食事で、野菜から先に食べてみる
- 寝る前にスクワットを10回やってみる
- 1日の終わりに「今日できたこと」を1つ書いてみる
- どれか1つで十分。「始めた」こと自体がすごいこと

**ステップ3：一人で抱え込まなくて大丈夫**
- ダイエットの情報は山ほどあるけど、自分に合うやり方は一人だと見つけにくいもの
- 「何を食べたらいいかわからない」「運動のやり方が合っているか確認したい」
- そんな方のために、STAY GOLDでは無料のオンライン相談を行っています
- あなたの体質・生活スタイルに合わせたアドバイスを、一対一でお伝えします

**CTA（行動喚起）**
- 「まずは気軽に相談してみる」→ 体験セッション予約リンク
- プレッシャーのない表現：「無理な勧誘は一切ありません。あなたの話を聞かせてください」

**締めの言葉**
あなたの体は、あなたが食べたもの、動いた分、考えたことでできている。
今日から少しずつ変えていけば、半年後の自分はきっと違う。
焦らず、自分のペースで。あなたなら絶対にできる。応援しています。

## ファクトチェック結果

本構成案で使用している主な主張について、エビデンスの確認結果を以下にまとめます。

### 食事関連

| 主張 | 確認結果 | 根拠 |
|------|---------|------|
| 極端な食事制限で基礎代謝が下がる | **正確** | 適応熱産生（adaptive thermogenesis）として確立された現象。長期的なカロリー制限で基礎代謝が低下する（Rosenbaum & Leibel, 2010, Int J Obesity） |
| タンパク質の摂取目安：体重1kgあたり1〜1.5g | **正確** | 日本人の食事摂取基準（2020年版）では成人の推奨量は0.9g/kg/日。ダイエット中は筋肉維持のため1.2〜1.6g/kgが推奨される（Phillips & Van Loon, 2011, J Sports Sci） |
| 脂質は1gあたり9kcal、タンパク質・炭水化物は4kcal | **正確** | アトウォーター係数として確立された値 |
| 鶏むね肉100gのタンパク質が約22g | **正確** | 日本食品標準成分表2020年版（八訂）に基づく。皮なし鶏むね肉100gあたりタンパク質23.3g |
| 卵1個のタンパク質が約6g | **正確** | 日本食品標準成分表に基づく。全卵1個（約60g）あたりタンパク質約7.4g。Mサイズ（約50g可食部）で約6.2g |
| 食べる順番で血糖値上昇がゆるやかになる | **正確** | 野菜を先に食べることで食後血糖値の上昇が抑制される（Imai et al., 2014, Diabetes Care） |
| 脂肪1kgは約7,200kcal | **正確** | 脂肪組織1kgは純粋な脂肪ではなく水分等を含むため、約7,200kcal（純脂肪1gの9kcal × 約800g）として広く使用される計算値 |

### 運動関連

| 主張 | 確認結果 | 根拠 |
|------|---------|------|
| 30分のジョギングで消費するカロリーは約200kcal | **概ね正確** | 体重55kgの女性が時速8kmで30分走った場合、約200〜230kcal程度。体重・速度で変動するため「約200kcal」「おにぎり約1個分」は妥当な表現 |
| 基礎代謝の20〜30%が筋肉で消費される | **正確** | 骨格筋は安静時代謝の約20〜25%を占める（Elia, 1992）。一部の文献では30%とするものもあり、20〜30%は妥当な範囲 |
| 30代以降、筋肉は年に約0.5〜1%ずつ減少 | **正確** | サルコペニアの進行として確立。30歳以降、年間0.5〜1%の筋肉量減少が報告されている（Volpi et al., 2004, J Gerontol） |
| 筋トレ後にカロリー消費が続く（EPOC） | **正確** | 運動後過剰酸素消費（Excess Post-exercise Oxygen Consumption）は確立された現象。高強度の筋トレ後、数時間〜最大72時間にわたり代謝が亢進する（Borsheim & Bahr, 2003） |
| 下半身の筋肉は体全体の約60〜70% | **正確** | 下半身の筋肉群（大臀筋、大腿四頭筋、ハムストリングス等）は体全体の筋肉量の約60〜70%を占めるとされる |

### 習慣・メンタル関連

| 主張 | 確認結果 | 根拠 |
|------|---------|------|
| 意志力は消耗する（自我消耗） | **正確（ただし議論あり）** | バウマイスターの自我消耗モデル（Baumeister et al., 1998）は広く知られるが、再現性について議論がある（Hagger et al., 2016の大規模追試）。ただし「意志力には限界がある」という実用的なメッセージとしては妥当 |
| ストレスホルモン（コルチゾール）と腹部脂肪の関係 | **正確** | 慢性的なコルチゾール上昇は内臓脂肪の蓄積と関連する（Epel et al., 2000, Psychosom Med）。コルチゾールは食欲増進にも関与 |
| 内発的動機の方が外発的動機より持続しやすい | **正確** | 自己決定理論（Deci & Ryan, 1985, 2000）で確立された知見。内発的動機づけは行動の持続性と正の相関がある |
| 健康的な減量ペースは月1〜2kg | **正確** | 多くの医療ガイドラインで週0.5〜1kg（月2〜4kg）を上限としているが、急激な体重減少のリスクを考慮すると月1〜2kgはより安全で持続可能な推奨値。厚生労働省も急激な減量を推奨していない |
| 体重は1日で1〜2kg変動する | **正確** | 水分摂取量、食事量、排泄などにより、1日の中で体重は1〜2kg程度変動するのは正常な範囲 |
